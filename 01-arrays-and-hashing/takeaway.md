# 🔥 Core Pattern: HashMap / Set

## Main Idea

Instead of repeatedly searching through arrays (`O(n)`),
store information while traversing so lookup becomes `O(1)`.

---

## ⚡ Common Triggers → Patterns

duplicates -	Set / HashMap

pair sum -	HashMap
uniqueness -	Set
